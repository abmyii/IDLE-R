IDLE-Remake
===========
A simple IDE aiming to be a remake of, and replacement to IDLE. Inspired by https://inventwithpython.com/blog/2014/11/20/idle-reimagined/.

Requirements
=============
PySide >= 1.2.1
pygments >= 2.1

Optional Requirements
=====================
qtconsole >= 4.2.0

Works with python-2.7+ and python-3.4+
